Principais formatações

Negrito / Destaque

Nesta frase, tenho um termo em negrito usando a tag <b>(não semântica).

Nesta frase, tenho um termo em destaque usando a tag <strong> (semântica).

Itálico / Ênfase

Nesta frase, tenho um termo em itálico usado a tag <i> (não semântica).

Nesta frase, tenho um termo em ênfase usando a tag <em> (semântica).

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>

Texto deletado e inserido

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Para fazer isso, usa a tag <del>

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois. Para fazer isso, usa a tag <ins>

Texto sobrescrito e subscrito

Para inserir coisa do tipo x20+3. Para fazer isso, usa a tag <sup>

Para inserir coisas do tipo H20. Para fazer isso, usa a tag <sub>
